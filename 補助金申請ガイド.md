## Mieru Counter — 申請先一覧と準備事項

## 1. 推奨する申請先（優先順）

### 1-1. IT導入補助金（最も申請しやすい）

| 項目 | 内容 |
|------|------|
| 管轄 | 経済産業省 / 中小企業基盤整備機構 |
| 補助率 | 1/2〜3/4 |
| 補助上限 | 最大450万円（通常枠） |
| 公募時期 | 例年3月〜12月（複数回の締切あり） |
| 申請先 | https://it-shien.smrj.go.jp/ |
| 該当枠 | **通常枠（A・B類型）** または **デジタル化基盤導入枠** |

**申請のポイント:**

- Mieru Counterを「ITツール」として登録する必要がある
- IT導入支援事業者として登録するか、既存の支援事業者と連携する
- 「業務効率化」「顧客サービス向上」の観点で申請書を書く
- 合理的配慮義務への対応ツールとして法的背景を強調する

**必要書類:**
- [ ] GビズIDプライムの取得（申請に2〜3週間かかるため最優先）
- [ ] 直近2期分の確定申告書/決算書
- [ ] 事業計画書（上記で作成済み）
- [ ] ITツール導入による業務改善効果の説明資料
- [ ] 見積書

### 1-2. ものづくり・商業・サービス補助金

| 項目 | 内容 |
|------|------|
| 管轄 | 経済産業省 / 全国中小企業団体中央会 |
| 補助率 | 1/2〜2/3 |
| 補助上限 | 最大1,250万円（通常枠） |
| 公募時期 | 通年（締切は年4回程度） |
| 申請先 | https://portal.monodukuri-hojo.jp/ |
| 該当枠 | **通常枠** または **デジタル枠** |

**申請のポイント:**

- 「革新的なサービスの開発」として申請
- 音声認識統合、AI重要事項抽出など**技術的な革新性**をアピール
- 「生産性向上」の数値目標が必要（付加価値額年率3%以上、給与支給総額年率1.5%以上）
- 事業計画の「将来性」と「市場規模」を明確に記載

**必要書類:**
- [ ] GビズIDプライムの取得
- [ ] 事業計画書（10ページ以内の要約版を作成）
- [ ] 直近2期分の確定申告書/決算書
- [ ] 賃金引上げ計画の表明書
- [ ] 認定支援機関の確認書

### 1-3. 小規模事業者持続化補助金

| 項目 | 内容 |
|------|------|
| 管轄 | 商工会議所 / 商工会 |
| 補助率 | 2/3 |
| 補助上限 | 最大200万円（賃金引上げ枠等） |
| 公募時期 | 年4回程度 |
| 申請先 | 地域の商工会議所 |
| 該当枠 | **通常枠** または **賃金引上げ枠** |

**申請のポイント:**

- 従業員20人以下の小規模事業者向け
- 「販路開拓」の観点で申請（新サービスの市場投入）
- 商工会議所の窓口で**無料で相談・申請支援**を受けられる
- 申請書のフォーマットが比較的シンプル

**必要書類:**
- [ ] 経営計画書（様式2〜3）
- [ ] 補助事業計画書（様式2〜3の一部）
- [ ] 直近の確定申告書
- [ ] 商工会議所の事業支援計画書（様式4）— 商工会議所が作成

### 1-4. 日本政策金融公庫 — 新創業融資制度

| 項目 | 内容 |
|------|------|
| 融資限度額 | 最大3,000万円（うち運転資金1,500万円） |
| 利率 | 年2.0〜2.5%程度（変動あり） |
| 担保 | 不要 |
| 保証人 | 不要 |
| 返済期間 | 設備資金20年以内、運転資金10年以内 |
| 申請先 | 最寄りの日本政策金融公庫支店 |

**申請のポイント:**

- 創業計画書が最重要（公庫所定の様式あり）
- **自己資金が融資額の1/10以上**あること
- 業界経験や該当分野の知識をアピール
- MVPが動作していることは非常に大きなプラス材料

**必要書類:**
- [ ] 創業計画書（公庫所定様式）
- [ ] 事業計画書（上記で作成済み — 補足資料として添付）
- [ ] 見積書（開発費、サーバー費等）
- [ ] 通帳コピー（自己資金の確認）
- [ ] 履歴書
- [ ] 不動産登記簿（事務所が賃貸の場合は賃貸借契約書）

## 2. 申請の進め方

### Step 1: 事前準備（1〜2週間）

```
□ GビズIDプライムを取得する（https://gbiz-id.go.jp/）
  → 書類郵送後、2〜3週間で発行
  → ほぼ全ての補助金申請に必要

□ 最寄りの商工会議所に相談予約を入れる
  → 無料で補助金の選定・申請支援を受けられる
  → 認定支援機関の紹介も受けられる

□ 確定申告書/決算書を用意する
  → 直近2期分が必要
  → 創業前の場合は不要（創業計画書で代替）
```

### Step 2: 認定支援機関への相談（1週間）

```
□ 認定支援機関（税理士・中小企業診断士）に相談する
  → ものづくり補助金は認定支援機関の確認書が必須
  → 事業計画書のブラッシュアップも依頼できる
  → 成功報酬型（補助金額の5〜10%程度）の機関もある

  認定支援機関の検索:
  https://ninteishien.force.com/NSK_CertificationArea
```

### Step 3: 申請書類の作成（2〜3週間）

```
□ 事業計画書を補助金の様式に合わせて編集する
  → 事業計画書.md をベースにする
  → 補助金ごとに求められる記載項目が異なる

□ 見積書を取得する
  → 開発費: 外注する場合は開発会社から
  → サーバー費: Vercel/Supabase/AWS等の料金表
  → 人件費: 従業員を雇用する場合の給与計算

□ デモ動画を作成する（推奨）
  → 実際の操作画面を録画（3〜5分程度）
  → 審査員の理解を助ける補足資料として添付
```

### Step 4: 申請（jGrants経由）

```
□ jGrants（https://jgrants2.go.jp/）にログイン
  → GビズIDでログイン
  → 申請フォームに入力
  → 添付書類をアップロード
  → 提出

□ 締切に注意
  → 締切日の直前はシステムが混雑する
  → 2〜3日前に提出完了することを推奨
```

### Step 5: 審査・採択（1〜2ヶ月）

```
□ 書類審査
  → 追加質問が来る場合がある — 迅速に回答する

□ 採択通知
  → 採択されたら交付申請を行う
  → 事業実施期間内に開発・導入を完了させる

□ 実績報告
  → 事業完了後に実績報告書を提出
  → 証拠書類（領収書、契約書、成果物）を保管しておく
```

## 3. 申請書のアピールポイント（共通）

以下の要素は、どの補助金・融資でも有効なアピール材料です。

### 社会的必要性

> 障害者差別解消法の2024年改正により、民間事業者にも合理的配慮の提供が義務化された。
> 全国の薬局約60,000店、医療機関約180,000施設は、聴覚障害者への情報保障を行う法的義務があるが、
> 手話通訳者の慢性的不足（全国約3,500人）や筆談の業務負担により、十分な対応ができていない。
> 本事業はこの社会的課題を、ITの力で低コストかつ即座に解決するものである。

### 技術的優位性

> 既存の音声認識アプリ（UDトーク等）と異なり、本サービスは窓口業務に特化しており、
> 重要事項カードの自動抽出、双方向テキスト通信、確認ボタンによる意思表示、
> スマートグラス連携など、窓口での情報保障に必要な機能を統合的に提供する。
> Webブラウザのみで動作し、専用ハードウェアが不要なため、導入コストが極めて低い。

### 事業の実現可能性

> MVPは既に開発完了し、本番環境（https://mieru-counter.vercel.app）で動作している。
> 技術的な実現可能性は実証済みであり、今回の申請では音声認識統合やAI連携など、
> サービスの品質向上と市場投入に必要な開発に資金を充当する。

### 波及効果

> 本サービスは薬局・病院だけでなく、役所・銀行・ホテルなど幅広い業種の窓口に
> 適用可能であり、業種別テンプレートの提供により横展開が容易である。
> また、多言語対応により外国人住民の行政手続き支援にも活用でき、
> 多文化共生社会の実現にも貢献する。

## 4. 費目の整理（申請書記載用）

補助金申請では、経費を費目ごとに分類する必要があります。

| 費目 | 内容 | 金額（税抜） |
|------|------|------------|
| ソフトウェア開発費 | 音声認識統合、AI連携、多言語対応の開発 | 500万円 |
| クラウド利用費 | サーバー、データベース、ドメイン（1年分） | 60万円 |
| 外注費 | UI/UXデザイン、アクセシビリティ監査 | 100万円 |
| 広報費 | Webサイト制作、営業資料、展示会出展 | 150万円 |
| 旅費 | パイロット施設への訪問、展示会参加 | 30万円 |
| 専門家経費 | セキュリティ監査、法務相談 | 50万円 |

> **注意:** 人件費は補助金によって対象外の場合があります。ものづくり補助金では対象外、
> IT導入補助金では「ITツールの導入費用」のみが対象です。
