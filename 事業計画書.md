## 1. 事業概要

### 1.1 事業の目的

Mieru Counter（ミエルカウンター）は、**聴覚障害者・難聴者・高齢者が窓口で会話内容を正確に理解できるようにする**ためのリアルタイム字幕・情報伝達サービスである。

薬局、病院、役所、銀行などの窓口において、スタッフの発話内容をリアルタイムで利用者の端末に字幕表示し、重要事項の確認、双方向のテキスト通信、確認ボタンによる意思表示を可能にする。

### 1.2 サービス内容

| 機能 | 説明 |
|------|------|
| リアルタイム字幕 | スタッフの発話・入力を利用者のスマホにリアルタイム表示 |
| 重要事項カード | 会話から重要な情報（薬の飲み方、手続き期限等）を抽出し、カード形式で表示・保存 |
| 双方向テキスト通信 | 話すことが難しい利用者がテキストでスタッフに伝達 |
| 確認ボタン | 「理解しました」「もう一度」等のワンタップ意思表示 |
| QRコード接続 | スマホでQRコードを読むだけでセッションに参加 |
| 定型文管理 | 業種ごとに頻出フレーズを事前登録・送信 |
| スマートグラス対応 | Even G2等のスマートグラスに字幕を表示（将来対応） |
| マルチ業種対応 | 薬局・病院・役所・銀行等、業種別のテンプレートを提供 |

### 1.3 提供形態

クラウド型SaaS（Software as a Service）として提供する。施設側はWebブラウザのみで利用開始でき、専用ハードウェアの導入は不要。

## 2. 事業背景と社会的課題

### 2.1 社会的課題

1. **聴覚障害者の窓口対応の困難**
   - 全国の聴覚障害者数：約36万人（身体障害者手帳所持者）
   - 難聴者を含む聞こえに困難を抱える人：推定**約1,400万人**（日本補聴器工業会調査）
   - 高齢化により難聴者は年々増加（65歳以上の約30%が難聴）

2. **窓口での情報格差**
   - 薬局での服薬指導、役所での手続き説明、病院での診察内容など、重要な情報が正確に伝わらない
   - 筆談は時間がかかり、スタッフの業務負担が大きい
   - 手話通訳者の慢性的な不足（全国で約3,500人）

3. **法的義務の強化**
   - **障害者差別解消法（2024年4月改正）**：民間事業者にも合理的配慮の提供が**義務化**
   - 薬局・病院・金融機関等は、聴覚障害者への情報保障を行う法的義務がある
   - 対応できていない施設は法令違反のリスクがある

### 2.2 既存の対応方法の限界

| 方法 | 課題 |
|------|------|
| 筆談 | 時間がかかる、双方向のやりとりが困難 |
| 手話通訳 | 人材不足、予約が必要、常駐は困難 |
| UDトーク等の既存アプリ | 窓口業務に特化しておらず、重要事項の抽出・確認機能がない |
| コミュニケーションボード | 定型的な内容しか伝えられない |

### 2.3 本事業の解決策

Mieru Counterは、既存の端末（PC・タブレット・スマホ）だけで動作し、**追加ハードウェア不要**で導入できる。リアルタイム字幕に加え、重要事項の抽出・確認、双方向テキスト通信、確認ボタンによる意思表示など、**窓口業務に特化した機能**を提供する。

## 5. 競合分析と差別化

### 5.1 競合サービス

| サービス | 特徴 | 限界 |
|---------|------|------|
| UDトーク | 音声認識字幕アプリ | 窓口業務特化ではない。重要事項管理なし |
| こえとら | 聴覚障害者向けコミュニケーションアプリ | 1対1前提。施設運用に不向き |
| SpeechCanvas | 筆談アプリ | リアルタイム性に欠ける |
| Google翻訳/音声入力 | 汎用音声認識 | 業務フローへの統合なし |

### 5.2 Mieru Counterの差別化ポイント

1. **窓口業務に特化**: セッション管理、定型文、呼び出し機能など窓口業務フローに統合
2. **重要事項カード**: 会話から重要情報を抽出し、利用者が見返せる形で保存
3. **双方向コミュニケーション**: 確認ボタンとテキスト入力で利用者も意思表示可能
4. **マルチ業種対応**: 業種別テンプレートで薬局・病院・役所・銀行に即座に対応
5. **導入ハードル最小**: Webブラウザのみで動作、専用機器不要
6. **スマートグラス対応**: 将来的にEven G2等に字幕表示し、自然な対面会話を実現

## 12. まとめ

Mieru Counterは、**障害者差別解消法の改正による合理的配慮の義務化**という社会的要請に応える、窓口特化型の聴覚バリアフリーサービスである。

MVPは既に完成し、本番環境で動作しており、いつでもデモ可能な状態にある。今後、音声認識・AI連携・スマートグラス対応を段階的に実装し、薬局から病院・役所・金融機関へと展開していく。

本事業は、社会的意義と市場性を兼ね備えた事業であり、補助金・融資を活用して早期に市場投入を目指す。
